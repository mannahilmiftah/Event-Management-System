# Event Management System (Java Application)

An **Event Management System / Event Planner** built in **Java** to support efficient planning and organization of events within an educational institute.

## Overview

This application helps users manage events end-to-end—starting from account access to creating events, scheduling, and managing participants—along with news/updates and account controls.

## Key Features

- **User Authentication** (Login & Registration)
- **Event Creation & Scheduling**
- **Participant Management**
- **News / Updates Module**
- **Account Management** (Update & Delete Account)

## Tech Stack

- **Java** (Core Application)
- **MySQL** (Database)
- **MySQL Connector** (Database connectivity)
